# Lord of the Rings Scraping

This artifact will look for "Lord of the Rings" characters in a wiki website with web scraping.

## Configuration

### Install dependencies

```console
foo@bar:~$ npm i
```

### Run in development

```console
foo@bar:~$ npm run dev
```

### Build and run

```console
foo@bar:~$ npm run build && npm run
```
